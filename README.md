# 📚 Advanced Learner Management System

A comprehensive, cloud-based learner management system designed for Comprehensive High School Bajja — Lukaya. Built with vanilla JavaScript and Supabase for real-time data persistence.

## 🌟 Features

### 📊 Core Functionality
- **Class Management**: Create, switch, and delete classes with custom class codes (e.g., S1, S2, S5A)
- **Learner Management**: Add, edit, delete, and search learners with auto-generated student IDs
- **Assessment Management**: Create assessments with customizable grading scales (CBC, UCE, Letter)
- **Mark Entry**: Record and track marks per learner per assessment with real-time updates
- **Analytics Dashboard**: Visualize class performance with charts and statistics
- **Individual Reports**: Generate detailed A-Level subject reports with grade calculations

### 🎯 Key Features
- **Auto-generated Student IDs**: IDs like `S501`, `S502` based on class codes
- **Multiple Grading Scales**: CBC Descriptor, UCE (D1–F9), Letter (A–F)
- **Weighted Performance**: Calculate overall scores with customizable assessment weights
- **PDF Generation**: Export assessment results and individual reports
- **CSV Export**: Export learner registers and full class data
- **Print Support**: Print reports directly from the browser
- **Data Backup/Restore**: Local JSON backup and restore functionality
- **Cloud Sync**: All data automatically synchronized with Supabase
